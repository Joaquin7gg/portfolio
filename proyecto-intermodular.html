Proyecto Intermodular ASIR

Diseño e implementación de una infraestructura IT completa para un club deportivo, planteado como un entorno pyme real.

Estado: desplegado Tipo: interdisciplinar Formación: ASIR Año: 2025
Ver repositorio ↗ ← Volver al portfolio
01

Resumen

El objetivo principal ha sido construir una solución tecnológica coherente, funcional y escalable para un club deportivo estructurado como pequeña-mediana empresa, con varios departamentos. La solución cubre las necesidades de los usuarios y de la administración: conectividad, almacenamiento, servicios corporativos y publicación de recursos.

Objetivo Integrar en un único entorno todas las piezas de una infraestructura real —hardware, red, sistemas, servicios de directorio, bases de datos y nube— y documentar el proceso de principio a fin.
02

Alcance e integración

Es un proyecto interdisciplinar que conecta hardware, redes, sistemas, bases de datos, lenguajes de marcas y cloud. Las piezas no van sueltas: la intranet se integra en el entorno de servidor, las bases de datos en la nube se conectan con la infraestructura local, los inventarios en XML documentan los activos de hardware y la red segmentada por VLAN soporta el despliegue de los servicios.

03

Arquitectura técnica

Hardware

  • Sobremesas, estaciones de trabajo y portátiles profesionales
  • Servidor principal con RAID 5
  • Switch multicapa y switches de acceso
  • Router y puntos de acceso WiFi
  • SAI (alimentación ininterrumpida)

Redes

  • Diseño en Cisco Packet Tracer
  • Segmentación por VLANs por departamento
  • Enrutamiento inter-VLAN con enlaces trunk
  • WiFi corporativa y de invitados separadas

Sistemas

  • Windows Server 2022 y Windows 11 Pro
  • Ubuntu Desktop y Kali Linux
  • Active Directory (usuarios y permisos)
  • DNS, servidor de archivos, impresión, FTP e IIS
  • Automatización con PowerShell

Bases de datos & Cloud

  • MySQL / MariaDB con phpMyAdmin
  • AWS (nivel práctica): EC2, S3, RDS, VPC
  • Servidor web Apache
  • Gestión con AWS CLI

Web & marcas

  • XML con validación DTD
  • Transformación con XSLT
  • Intranet en HTML / CSS
04

03
